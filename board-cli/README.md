# 仮想役員会CLI

相談を投げると、立場(レンズ)の違う5人の役員が議論して、結論を **1つ** に絞って出すCLI。
**Google Gemini の無料枠**で動きます。

- 役員の発言も最終まとめも `gemini-2.5-flash`（無料枠で動く）
- 役員ラウンドは並列で叩き、無料枠の `429`（上限）は**自動で待って再試行**
- 有料プランなら `GEMINI_SUMMARY_MODEL=gemini-2.5-pro` でまとめだけ高性能化も可
- 役員は「役割(レンズ)」で定義。名前はただのラベルで、**本人の実発言は再現しない**
- 数字・固有事実には必ず `[要裏取り]` を付けさせる

## セットアップ

```bash
cd board-cli
npm install

# Google AI Studio で無料のAPIキーを取得 → https://aistudio.google.com/apikey
export GEMINI_API_KEY="AIza..."
```

> 無料枠にはレート制限（1分あたり/1日あたりのリクエスト数）があります。
> 役員ラウンドは5本を並列で叩くので、上限に当たったら少し待つか、
> まとめも flash にすると軽くなります（`export GEMINI_SUMMARY_MODEL=gemini-2.5-flash`）。

## 使い方

```bash
# 標準の役員会
node index.js "新規事業に1000万投資すべきか"

# board を切り替える（値上げ用）
node index.js -b neage "主力プランを20%値上げすべきか"

# 前提を差し込んで実行（採用用）
node index.js -b saiyo -p "現場の人数が足りていない" "シニアを採るべきか"

# board 一覧
node index.js --list

# 対話モードに入らず1回だけ
node index.js --once "..."
```

`npm install` 後は `npx board "..."` でも起動できる（`bin` 登録済み）。

## 議論フロー（一問一答にしない）

1. **各役員が自分のレンズで意見を出す**（flash・並列）
2. **他4人の意見を渡して、お互いに反論させる**（flash・並列）
3. **懐疑派役に「最大の懸念を1つだけ」言わせる**
4. **全議論を gemini-2.5-pro に渡し「私が今日決めるべきことを1つに絞って」で結論を強制**

出力は端末に `各役員の意見 → 反論 → 最大の懸念 → 最終結論` の順。

## 対話モード

1回実行したあと、追加の前提を打ち込むと、それを反映して「もう一回」議論できる。

```
前提を追加 > 現場の人数が足りてない前提で再実行
```

コマンド:

| 入力 | 動作 |
|------|------|
| `:board <name>` | board を切り替える（例 `:board saiyo`） |
| `:list` | board 一覧 |
| `:reset` | 追加前提をすべてクリア |
| `:show` | いまの前提を表示 |
| `:quit` / 空Enter | 終了 |
| それ以外 | すべて「追加前提」として再実行 |

## board の定義

`boards/<name>.json` に保存。`--board <name>` で選択（パス指定も可）。

```jsonc
{
  "name": "default",
  "title": "標準役員会",
  "description": "汎用。新規施策の意思決定全般に。",
  "members": [
    {
      "id": "investor",
      "label": "孫正義",          // 名前は「ラベル」扱い（本人ではない）
      "role": "攻めの投資家",      // 役割 = 思考のレンズ
      "tone": "断定的で大胆",      // 口調
      "priority": "市場を取れるか最優先", // 最優先する価値観
      "skeptic": false            // true にすると悪魔の代弁者役
    }
    // ... 計5名
  ]
}
```

同梱の board:

| name | 用途 |
|------|------|
| `default` | 汎用・新規施策の意思決定全般 |
| `saiyo` | 採用判断 |
| `tettai` | 撤退判断 |
| `neage` | 値上げ判断 |

新しい board を増やすときは `boards/` に `<name>.json` を足すだけ。

## 役員の制約（固定）

各役員の system prompt には次が固定で入る:

> お前は思考のレンズであって本人ではない。本人が実際にこう言ったと書くな。

加えて、数字・固有事実を出したら直後に `[要裏取り]` を付けるよう強制している。
