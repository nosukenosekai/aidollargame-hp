## ✍️ 投稿のしかた（コピペ3分）

1. note にログイン → 右上「**投稿**」→「**テキスト**」
2. 原稿の**1行目（タイトル）**をタイトル欄に貼る
3. **残りの本文**を本文欄に貼る
4. 見出し・太字を設定（下のルール参照）
5. アイキャッチ画像 → ハッシュタグ → 「**公開**」

### 書式のルール
原稿内の表記を、note の編集画面でこう設定してください：

- `## ◯◯◯` の行 → note の「**見出し**」に設定（目次が自動で出ます）
- `**◯◯**` で囲まれた部分 → note の「**太字（B）**」に設定
